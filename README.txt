+++ EXPLOSIVE PONG +++

Explosive Pong is a game based on the arcade classic Pong that is based around the idea of aiming at targets, applying different status effects on the ball, and even blow it up to score multiple hits.

How to start it:
Open src/html/explosive_pong.html on a local server using the Visual Studio Code Live Server Extension.
If you simply open it in your browser withput the Live Server, it will not work.
